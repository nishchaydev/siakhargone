# **App Name**: SIA Khargone

## Core Features:

- Dynamic Homepage: Showcase school highlights, sections, announcements, and latest updates dynamically.
- AI Chatbot: AI-powered chatbot in the bottom-right corner to assist parents with admissions, facilities, timings, and contact details.
- Admissions Portal: Step-by-step guide for admissions with a detailed explanation of the process and a prominent 'Apply Now' button linked to an enquiry form.
- Virtual Campus Tour: Interactive virtual tour using 360Â° images or embedded video to showcase school infrastructure and facilities.
- Firestore Integration: Utilize Firestore to store and manage dynamic data such as academic programs, events, gallery items, and chatbot responses.
- Image Gallery: Dedicated gallery page displaying school photos, events, and campus visuals, automatically fetched from Firestore or Firebase Storage.
- Contact Form: Functional contact form allowing parents to submit queries directly, stored in Firestore or emailed via Firebase Functions.

## Style Guidelines:

- Primary color: #1E3A8A (Navy Blue)
- Background color: #F2F2F2 (Off-white)
- Accent color: #EAB308 (Gold)
- Primary font: Poppins, sans-serif
- Use clean, modern icons related to education, learning, and school life.
- Responsive design
- Mobile-first design
- Fade-in / Slide-up for page transitions
- Soft elevation on hover for cards and buttons
